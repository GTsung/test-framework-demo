# test-framework
    
本项目为测试框架调研demo，列举了常用的测试框架基础用法。

~~~ 
   工程所用sql在sql目录下。
~~~

### 工程结构
* common-demo: 公共依赖
* junit5-demo: 基于JUnit5的测试
* testng-demo: 基于TestNG的测试
* mockito-demo: 基于Mockito的测试
* power-mockito-demo: 基于PowerMockito的测试
